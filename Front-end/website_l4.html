Lesson 4: Understanding URLs

A URL (Uniform Resource Locator) is like the address of a webpage, telling your browser where to go to retrieve information.

Breaking down a URL:

  • Scheme (http: or https):

    Indicates how to access the resource. Secure websites use HTTPS.

  • Domain Name (www.example.com):

    The main identifier for a website, typically representing the brand.

  • Subdomain (www.):

    Often included by default, used for specific sections like blog.example.com.

  • Top-Level Domain (TLD) (.com):

    The ending part of the domain, like .com, .edu, or country-specific TLDs.

  • File Path (/popular/clothes):

    Directs users to specific pages or sections within the website.

  • Parameter String (?title=clothes):

    Passes extra information to load content dynamically.

  • Anchor (#dresses-women):

    Jumps to a specific part of the webpage.
