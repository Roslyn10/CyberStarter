Lesson 2: How the Internet Works

The internet connects to servers that store the data/websites we want to access.

To connect, you need an ISP (Internet Service Provider) and a router.

When you enter a website address, your request goes to the ISP, then to the DNS (Domain Name System), and finally to the server hosting the website.

Data travels through internet cables in packets, which are reassembled at the destination to display the webpage.

Routers direct these packets based on IP addresses, following rules called the Transmission Control Protocol (TCP), which ensures smooth data transmission.
